# Figure rules

## Figure sources

Allowed sources:

1. User uploaded image
2. User supplied data plot
3. Redrawn method diagram based on cited source
4. AI generated concept diagram
5. Placeholder for user replacement

## AI generated figures

AI generated figures may only be used for concept diagrams, process diagrams, architecture sketches, or visual explanation.

They must not be used as experimental results, real screenshots, real dataset samples, or paper figures.

## Paper figures

If using a paper figure directly, cite the source and check whether the assignment allows it.

If redrawing a paper figure, state that it is adapted from the source.

## Experimental figures

Experimental plots must come from user supplied data, experiment logs, or real results.

Never invent values to make a plot.

## Figure placement

Every figure must have:

- Figure number
- Caption
- Source status
- Mention in main text
- Target section

## Figure captions

A caption should identify what the figure shows and why it matters. It should not make claims that are not supported by the figure.
