# Citation rules

## General rules

Use real sources only. Do not invent titles, authors, venues, years, DOIs, URLs, or page numbers.

If a needed source is missing, add a clear placeholder and ask the user for the source.

Every factual claim from papers, books, slides, datasets, or web sources needs citation unless it is common background knowledge.

## Default styles

For Chinese course reports, use GB/T 7714 when no style is given.

For English engineering reports, use IEEE when no style is given.

Use APA only if the course asks for it or the report belongs to a field where APA is expected.

## GB/T 7714 behavior

Use numbered citations in text when required. Format the reference list according to GB/T 7714 conventions.

Common types include journal article, conference paper, book, dissertation, report, standard, and online resource.

## IEEE behavior

Use numbered citations such as [1], [2], and [3]. The reference list order follows first appearance in the report.

## Source quality

Prefer user supplied course sources and assigned papers.

Do not add extra web sources unless the user asks for more references or current information is required.

## Citation placeholders

Use placeholders such as `[待补充来源：方法定义]` or `[source needed: dataset description]` when the source is not available.
