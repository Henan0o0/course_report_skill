# Final review checklist

Run this checklist before final delivery.

## Assignment match

- Topic matches the assignment
- Report type matches the assignment
- Length is within range
- Language matches the request
- Required sections are present
- Required template is followed
- Teacher instructions are followed

## Structure

- Title is specific
- Abstract matches the body
- Introduction states the problem and goal
- Related work is grouped logically
- Method section is clear
- Results or discussion section is supported
- Conclusion follows from the body
- References are present if needed

## Evidence

- No invented papers
- No invented data
- No invented experiment results
- No unsupported claims
- Missing sources are marked
- User supplied materials are used correctly

## Style

- Style profile is applied if provided
- Paragraph length is consistent
- Terminology is consistent
- Abbreviations are defined once
- Transitions are clear
- Tone matches course report style

## Figures

- Every figure has a number
- Every figure has a caption
- Every figure is mentioned in the text
- Figure source is clear
- AI generated figures are not used as real results
- Experimental plots are based on real data

## Formulas

- Formula notation is consistent
- Variables are explained
- Complex formulas are display formulas
- Formula numbering is consistent if required
- Word equations render correctly if Word output is requested

## Citations

- Citation style matches the request
- In text citations match the reference list
- Reference order is correct for numbered styles
- No fake references are included
- Placeholder citations are clearly marked

## Word output

- Headings render correctly
- Table of contents works if included
- Figures are visible
- Tables are readable
- Formulas render correctly
- Captions and numbering are consistent
- No unresolved placeholders remain unless reported
