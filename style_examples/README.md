# Style examples

Place previous course reports in this folder if they should be packaged with the skill.

Recommended use:

1. Store only reports that the user wants to use as style references.
2. Do not treat these files as factual sources for a new report.
3. Extract a style profile before drafting.
4. Do not copy sentences or paragraphs from these reports.

For privacy, remove personal information, teacher names, student IDs, grades, and private comments before packaging old reports.
