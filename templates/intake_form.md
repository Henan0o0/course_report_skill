# Course report intake form

Use this form at the start of a report task. Ask only for missing items.

## Task requirements

- Course or topic
- Report type
- Chinese or English
- Word count or page count
- Required template
- Required sections
- Grading rubric
- Submission format
- Teacher instructions

## Template

- User has a template
- User has no template but has formatting rules
- User has no template and accepts the default template

## Previous reports for style

- Use previous reports as style references
- Do not use previous reports
- Style strength
  - light
  - medium
  - strong

## Source materials

- Course slides
- Papers
- Textbook chapters
- Dataset
- Experiment logs
- Code
- Draft text
- Teacher notes
- User notes

## Sections

Default sections are abstract, introduction, related work, method, experimental results, conclusion, and references.

Record any section changes requested by the user.

## Figures

For each figure, record:

- Figure name
- Figure type
- Target section
- Source
- Caption
- Need AI generation
- Need user upload
- Need data plotting

## Formulas

- Formula needed
- Formula source
- Formula numbering needed
- Word equation output required

## Citations

- GB/T 7714
- IEEE
- APA
- Other
- No strict style given
