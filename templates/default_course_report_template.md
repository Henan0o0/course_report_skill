# Default course report template

Use this template only when the user has no required template.

## Chinese technical report

# 标题

## 摘要

简要说明研究背景、问题、方法、结果或分析结论。

## 关键词

关键词 1；关键词 2；关键词 3

## 1 引言

说明问题背景、研究意义、课程关联、报告目标和章节安排。

## 2 相关工作

按主题组织已有方法，不要只按论文逐篇堆叠。说明每类方法的思想、优点、限制和与本文任务的关系。

## 3 方法

说明本文关注的方法、模型、算法、公式、流程或系统结构。复杂公式放在行间，并解释变量含义。

## 4 实验结果

说明实验设置、数据、评价指标、结果图表和分析。若没有实验，将本节改为分析与讨论。

## 5 总结

总结报告主要内容、课程收获、方法限制和可能的改进方向。

## 参考文献

按用户指定格式列出真实来源。

## English technical report

# Title

## Abstract

Summarize the background, problem, method, findings, and conclusion.

## 1 Introduction

Explain the background, motivation, course connection, report goal, and structure.

## 2 Related Work

Group prior work by technical theme. Compare assumptions, strengths, limits, and relation to the report topic.

## 3 Method

Explain the method, model, algorithm, equations, pipeline, or system design.

## 4 Experimental Results

Explain setup, data, metrics, results, figures, and analysis. If there are no experiments, use Analysis and Discussion.

## 5 Conclusion

Summarize the main points, course learning, limitations, and future work.

## References

List checked sources in the chosen citation style.
