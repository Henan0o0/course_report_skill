# Outline confirmation template

Use this structure when asking the user to confirm the outline.

## Report brief

- Topic
- Report type
- Language
- Length
- Template
- Citation style
- Style reference status
- Formula status
- Figure status

## Proposed title options

1. Title option 1
2. Title option 2
3. Title option 3

## Proposed section plan

For each section, provide:

- Section title
- Writing goal
- Main points
- Required evidence
- Planned figures
- Planned formulas
- Planned citations
- Missing user inputs

## Figure plan

Use a table with figure index, figure name, source, target section, caption draft, and status.

## Formula plan

Use a table with formula index, formula purpose, source, target section, numbering, and Word export note.

## Citation plan

List expected citation positions and source status.

## Assumptions

List defaults and assumptions made due to missing information.

## User confirmation request

Ask the user to confirm, delete, rename, or reorder sections. Do not start full drafting unless the user confirms or asks to skip confirmation.
